<h2 align="center">Installation</h2>

```ini
[p]load downloader
[p]repo add shogun-cogs https://github.com/Project-MEI/shogun-cogs/
[p]cog install shogun-cogs <cog_name>
```
<sup>[p] is your prefix</sup>
